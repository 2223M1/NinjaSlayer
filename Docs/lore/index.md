# Ninja Slayer 原著知识库

本目录保存可版本管理的书目结构、检索别名和设计向知识。三部 EPUB 的正文只抽取到 Git 忽略的 `.lore-cache/`，不进入本目录。

## 入口

- [书目与章节索引](library-index.md)：源文件哈希、OPF/NCX 结构、章节与私有检索块。
- [`aliases.json`](aliases.json)：中英文、译名与异体写法的确定性检索扩展。
- [核心人物与关系](characters.md)：首轮只整理对 Ninja Slayer 模组设计直接有用的人物。
- [招式与装备](techniques-and-equipment.md)：茶道、风林火山、空手道、手里剑、受身和奈落之力等。
- [卡牌设计证据](design-evidence.md)：从原文事实到机制方向的显式推导。
- [不确定项](uncertainties.md)：尚未核实或不能安全泛化的细节。

## 标记与引用

- `[原文事实]`：可由紧随其后的原文位置直接支持。
- `[设计推论]`：从事实出发的模组设计解释，不属于设定事实。
- `[不确定]`：现有命中不足以定论，或不同语境可能同名异义。

统一引用格式为 `[第一部｜章节名｜p1-c003-s02.md:42]`。行号属于当前源 EPUB 哈希；换书或重建后要复核引用。

## 使用边界

先查索引和别名，再运行 `search_lore.py`，只阅读命中上下文；宽泛结论应再找第二处证据。回答以概述与短语核验为主，不大段复现原著。
