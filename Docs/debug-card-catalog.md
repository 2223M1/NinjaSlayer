**数值格式**：基础 → 升级（无变化则只写一项）。X 费牌单独标注。

**收录规则**：只统计 `NinjaSlayerDebugCardCatalog.CreateCards()` 实际返回的卡牌。Token 与 Status 可在战斗中生成，但不属于调试版本奖励卡池，因此不计入总数。

**调试池结构**：初始卡、20 张白卡和3张先古卡构成基线；蓝卡与金卡来自 `AdditionalCards`，可在同一清单中独立增删或替换，不影响正常版本卡池。

## 手里剑

### 白

备镖/ShurikenStock 白/技能 费用1
获得8点格挡。
蓄势1→2张手里剑。

投掷苦无/ThrowKunai 白/攻击 费用1
造成9→11点伤害。
预见2→3。
丢弃1张牌。

投剑/ShurikenThrow 白/攻击 费用1
造成10→12点伤害。
将1张手里剑 → 手里剑+洗入抽牌堆。

飞镖/ShurikenSpread 白/技能 费用1
将2张手里剑 → 手里剑+加入手牌。

### 蓝

咿啊铁斩波/IyaIronSlashWave 蓝/技能 费用1
你在本回合内不能再抽任何牌。
打出手里剑时，获得等同于其造成伤害的格挡。
消耗。
升级获得保留。

备刃/ReadyBlade 蓝/技能 费用1
蓄势3→4张手里剑。
抽1张牌。

手里剑大劈/ShurikenCleave 蓝/攻击 费用2
对所有敌人造成10→12点伤害。
你的手牌中每有1张手里剑或强·手里剑，伤害增加4→6。

搓手手/RubHands 蓝/攻击 费用1
造成6→8点伤害。
你在本回合中每打出过一张非手里剑攻击牌，将一张手里剑 → 手里剑+加入你的弃牌堆。

无星之夜/StarlessNight 蓝/能力 费用2→1
每当你生成手里剑时，消耗所有手里剑，并在你的手牌中加入1张强·手里剑。

### 金

剑来/BladesCome 金/能力 费用2→1
在你的回合开始时，在你的手牌中加入1张手里剑。

地狱龙卷/HellTornado 金/技能 费用3
获得翱翔。
在下个回合开始时，将所有手里剑打出2次。
翱翔时，手里剑会攻击所有敌人。
升级获得保留。
消耗。

## 奈落

奈落形态：进入时获得12层奈落生命；每当你攻击时，所有敌人失去4点生命；每打出一张攻击牌，将1张燃烧加入弃牌堆。

### 白

燃烧打击/BurningStrike 白/攻击 费用1
造成14→18点伤害。
将一张燃烧放入抽牌堆。

### 金

奈落回血/NarakuRecovery 金/技能 费用2
化身奈落。
获得10→12层奈落生命。
获得10→12点格挡。
消耗。

赤黑/RedBlackFlame 金/技能 费用2→1
化身奈落。
手中的所有攻击牌免费打出。
打出时消耗。

### 先古

一心同体/OneBodyOneSoul 先古/能力 费用3→2
虚无。
本场战斗不能再进入奈落；每次试图进入奈落，改为获得2点能量，抽2张牌，并获得12点奈落生命。
