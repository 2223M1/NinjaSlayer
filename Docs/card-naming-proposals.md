# Ninja Slayer 全卡牌命名采用记录

> 覆盖范围：当前 `card-catalog.md` 登记的全部 93 张牌。
> 本文记录已采用的中文显示名；不改类名、内部 ID、卡牌行为或数值。
> 命名规则：原著专名或固定短句优先，超过六个汉字时克制缩写；无合适原著词时使用《杀戮尖塔》原版风格。

## 1. 候选规则

- **原著/术语**：只使用原文出现过的招式、短语或非常克制的压缩。带 `≈` 的名称是根据原文意象缩写，不冒充固定招式；`—` 表示没有可靠直引。
- **机制直观**：直接说明这张牌最值得玩家记住的动作、触发条件或收益。
- **原版风格**：参考《杀戮尖塔 2》中文卡名的做法，以 2～6 个字的动作、状态或意象命名，不逐字复述规则。参考源为本地原版 `Slay the Spire 2/localization/zhs/cards.json`，例如“刀刃之舞”“先制打击”“后空翻”“完美打击”。
- **采用名**：原著专名或固定短句优先；同一原著词只交给效果最贴切的一张牌，其余采用互不重复的原版风格名。
- 初始牌“打击”“防御”保留原版基础牌惯例。Token 与 Status 优先采用稳定的机制名，避免玩家误判其行为。

## 8. 采用结果

- 93 张卡均有唯一采用名；卡池中无重复显示名。
- 保留原名：直拳、无星之夜、强·手里剑、完美chop、龙卷拳；基础牌“打击”“防御”与状态“茶道”也保持稳定。
- 原著术语缩写：“太古暗杀术茶道”→“太古暗杀术”。
- 原著长名压缩：“全方向无情投掷”→“全方向投掷”。
- 长原文候选改用原版风格：“茶道、然后再茶道”→“豪饮醒神”，“茶道、风林火山、茶道”→“入定不息”。
- `PerfectChop` 按类型名且忽略大小写统计 Chop 牌；显示名变更不影响机制。
